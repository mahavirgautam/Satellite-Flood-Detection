## 🌊 Project Overview

* 🚀 Built a **real-time flood monitoring system**
* 🛰️ Uses **Satellite Data (Sentinel-1 & Sentinel-2)**
* 🤖 Integrated **Deep Learning + LLM**
* 📊 Provides **flood maps, rainfall prediction & alerts**
* 🎯 Designed for **disaster management & decision support**

---

## ⚡ Key Features

* 🌍 Flood Detection using **SAR + NDWI**
* 🧠 **U-Net Model** for flood segmentation  
* 🌧️ **LSTM Model** for rainfall prediction  
* 🔗 **Data Fusion** (SAR + Optical data)  
* 💬 **AI Alert Generation (Flan-T5 / GPT)**  
* 📊 Interactive **Dashboard Visualization**

## 🧠 How It Works

🔹 **1. Data Collection**
- Sentinel-1 (SAR) → Works in clouds  
- Sentinel-2 → Water detection (NDWI)

🔹 **2. Flood Detection**
- U-Net generates flood masks  

🔹 **3. Rainfall Prediction**
- LSTM predicts rainfall patterns  

🔹 **4. Alert Generation**
- AI converts results into simple alerts  

🔹 **5. Visualization**
- Dashboard shows maps + alerts
