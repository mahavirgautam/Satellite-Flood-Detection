## 🌊 About The Project

🚀 This project is a **smart flood monitoring and alert system** that combines:

- 🛰️ Satellite Data (Sentinel-1 & Sentinel-2)  
- 🤖 Deep Learning Models (U-Net, LSTM)  
- 🧠 AI-generated alerts (Flan-T5 / GPT)  

👉 Designed to provide **real-time flood detection, rainfall prediction, and decision support**
